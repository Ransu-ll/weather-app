# weather-app
A simple Australian-focused weather webapp, for deployment as part of a certain university assignment.

# Getting started (the webapp)
1. Clone this repository
2. CD into cloned repo
3. Create a new environment inside webapp via `py -m venv .\webapp\env`
4. Activate the environment via `.\webapp\env\Scripts\activate`
5. Install the required modules via `pip install -r .\webapp\requirements.txt`
